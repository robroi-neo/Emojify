# Emojify: Facial Expression Classification

## What is this project about?
This project is for a Computer Vision class wherein we classify facial expressions into 7 different emojis.

---


## Creating a Conda Env

```conda create --name Emojify python=3.9 -y```
```conda activate Emojify```

## Installing Packages
```conda install pandas numpy matplotlib scikit-learn seaborn```

## Installing OpenCV
```conda install -c conda-forge opencv```

## Installing Tensorflow 
```conda install tensorflow```

## Installing jupyter
```conda install notebook```

## Exporting Env to YAML file
```conda env export --no-builds > environment.yml```

## Activate Recreated Environment
```conda activate your_env_name```
